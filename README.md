# Flipkart Clone Frontend

A simple Flipkart-style homepage built using **HTML**, **CSS**, and **Bootstrap 5**.  
It includes a working image carousel, product cards, and responsive design.

## Features
- Responsive layout using Bootstrap
- Auto-sliding image carousel
- Product cards with hover effects
- Gradient styled buttons
- Fixed navigation bar

## Technologies Used
- HTML  
- CSS  
- Bootstrap 5
